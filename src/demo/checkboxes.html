About Checkboxes

NOTE: Checkboxes will require Javascript. Below is an example if you want to use straight Javascript in order to make it work: 

                        $('.wwcode-square').click(function(){
                          var isChecked = event.target.nextElementSibling.checked;
                          $(this).toggleClass("selected");
                          event.target.nextElementSibling.checked = !isChecked;
                        });
